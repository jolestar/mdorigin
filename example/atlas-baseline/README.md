# Atlas Baseline

This example intentionally uses the built-in default presentation with no custom layout selection.

```json
{
  "siteTitle": "Example Site",
  "footerNav": [
    { "label": "GitHub", "href": "https://github.com/example/repo" }
  ]
}
```

Use this as the starting point for a site that only needs metadata, navigation, and light `stylesheet` overrides.
