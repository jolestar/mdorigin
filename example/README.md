# Examples

These examples show how to customize `mdorigin` without relying on built-in theme or template variants.

- `atlas-baseline/`: the default built-in presentation with no extra layout code
- `gazette-style/`: a stylesheet-only variant that recreates a publication-like tone
- `listing-layout/`: a `renderPage` example for custom listing pages

Treat these as copyable starting points, not as stable product APIs.
